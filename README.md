# TaskHub (Expo + React Native + Expo Router)

Projekt startowy spełniający wymagania:
- min. 3 ekrany + nawigacja (Expo Router: Stack + Tabs)
- interakcja z użytkownikiem (formularze, przyciski)
- autoryzacja (logowanie/rejestracja lokalna)
- przechowywanie danych (AsyncStorage)
- komunikacja sieciowa (fetch z API: JSONPlaceholder)
- lista elementów (FlatList)
- responsywny layout (flex)

## Uruchomienie

1. Zainstaluj zależności:
```bash
npm install
```

2. Start:
```bash
npx expo start
```

## Widoki
- /(auth)/login – logowanie
- /(auth)/register – rejestracja
- /(tabs) – zakładki: Zadania, Dodaj, Profil
